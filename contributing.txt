# Contributing to Lexgen

Thanks for your interest in improving Lexgen!
Lexgen is an open, YAML-powered CLI framework for deploying vertical AI agents. You can contribute:
- New agent templates (e.g. healthcare, finance)
- CLI features (e.g. `--dryrun`, `--trace`)
- Docs or examples

---

## âœ¨ Contributing an Agent Template

1. Fork the repo
2. Create your agent file:
```bash
lexgen template --init your-agent-name
```
3. Fill out the YAML with fields like:
```yaml
agent: policy-analyzer
vertical: insurance
task: summarize claim language
billing_unit: policy
format: pdf
```
4. Test it:
```bash
lexgen run --template policy-analyzer.yaml
```
5. Submit a PR with:
- Your YAML file in `/templates/`
- Example output in `/outputs/` (optional)
- Explanation in PR description

## ðŸ§© CLI Development
Want to improve or extend the CLI?
- Commands live in `lexgen/cli.py`
- Core router: `__main__.py`
- Logs live in `/logs/`, registry in `agent-registry.json`

Run local CLI:
```bash
pip install -e .
lexgen --help
```
